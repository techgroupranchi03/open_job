# Dart Console Debugging with Colors
```
  # Red text
 print('\x1B[31mRed: isSearching = $isSearching\x1B[0m');

  # Yellow text
  print('\x1B[33mYellow: isSearching = $isSearching\x1B[0m');

        # Green text
  print('\x1B[32mGreen: isSearching = $isSearching\x1B[0m');

  # Pink/Magenta text
  print('\x1B[35mPink/Magenta: isSearching = $isSearching\x1B[0m');

  # White text
  print('\x1B[37mWhite: isSearching = $isSearching\x1B[0m');
```
